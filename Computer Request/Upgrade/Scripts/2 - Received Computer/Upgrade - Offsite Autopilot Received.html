Now that you have your computer, we can start getting it set up! The first thing I will need is the serial number of your new device. This is required so I can ensure it is enrolled in Autopilot correctly.

 

Autopilot is a tool we use to automate the setup of your new computer. Once I verify that Autopilot is ready, you can set it up by following KB0015099 - Windows Autopilot. 

 

After that you have 2 weeks to migrate your data to your new computer, set up your OneDrive and reinstall software onto your new computer, then ship the old computer back.

Please let me know if you have any questions.
